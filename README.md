# Recu-ProgIV

Proyecto final de Programación IV: Aplicación Django para gestionar alumnos y scraper.

## Instalación
1. Clonar el repositorio
2. Crear y activar el virtualenv
3. Instalar dependencias con `pip install -r requirements.txt`
4. Ejecutar migraciones con `python manage.py migrate`
5. Ejecutar el servidor con `python manage.py runserver`
